|й–ұи®ҖйҮҸ:

жҲ‘д»ҘеүҚе…¶еҜҰжІ’жңүCI/CDзҡ„з¶“й©—пјҢжүҖд»ҘзҸҫжңүеёёиҰӢзҡ„CI/CDе№іеҸ°е…¶еҜҰйғҪжІ’жңүеӨӘйҒҺдәҶи§ЈпјҢдҪҶиҝ‘жңҹеӣ е°ҲжЎҲйңҖжұӮпјҢйңҖиҰҒйҖҸйҒҺGithub ActionгҖҖдҫҶе»әз«ӢдёҖеҖӢз”Ёж–јAzureиӘҚзҹҘжңҚеӢҷдёӯзҡ„ж–·еҸҘAPI (BreakSentence API) зҡ„CI/CD PipelineпјҢеңЁиў«дәӨд»ҳзҡ„йңҖжұӮйӮ„еҫҲжЁЎзіҠзҡ„зӢҖжіҒдёӢрҹҳӯпјҢйӮ„жҳҜе…ҲдҫҶдәҶи§ЈйҖҷй …еҠҹиғҪпјҢдёҰдё”еҒҡеҖӢзҙҖйҢ„гҖӮ

еҹәжң¬д»Ӣзҙ№

жҳҜдёҖеҖӢCI/CDе№іеҸ°пјҢиғҪеӨ иҮӘеӢ•еҢ–е»әзҪ®(Build)гҖҒжё¬и©Ұ(Test)д»ҘеҸҠйғЁзҪІ(Deployment)пјҢдҪҝз”ЁиҖ…еҸҜд»Ҙе»әз«Ӣе·ҘдҪңжөҒзЁӢ(Workflow)дҫҶе»әзҪ®д»ҘеҸҠжё¬и©ҰжҜҸеҖӢе°ҚRepositoryзҡ„pullи«ӢжұӮжҲ–иҖ…жҳҜе°ҮеҗҲдҪөзҡ„pullи«ӢжұӮйғЁзҪІеҲ°з”ҹз”ўз’°еўғгҖӮ

е…Ҳи¬ӣйҖҷдәӣе…ғд»¶зҡ„еҗҲдҪңиө·дҫҶзҡ„иЎҢзӮәжңғжҳҜжҖҺйәјжЁЈ: з•¶Repositoryдёӯзҷјз”ҹжҹҗеҖӢ дәӢд»¶(Event)пјҢжӯӨжҷӮдҪ жүҖиЁӯе®ҡзҡ„ Github Actions Workflows иў«и§ёзҷјгҖӮйҖҷиЈЎзҡ„дәӢд»¶пјҢиҲүдҫӢдҫҶиӘӘеҸҜд»ҘжҳҜдёҖеҖӢpull requestи«ӢжұӮжҲ–жҳҜжҹҗеҖӢissuedиў«е»әз«Ӣ зӯүзӯүгҖӮ

дҪ жүҖе®ҡзҫ©зҡ„Workflowз•¶дёӯеҢ…еҗ«дәҶдёҖеҖӢжҲ–еӨҡеҖӢ Jobs пјҢйҖҷдәӣJobsеҸҜд»ҘжҳҜиў«е№іиЎҢиҷ•зҗҶ(Parallel) жҲ–жҳҜ дҫқеәҸиҷ•зҗҶ(Sequential) пјҢжҜҸеҖӢJobsжңғеңЁиҮӘе·ұзҡ„иҷӣж“¬ж©ҹRunnerиў«иҷ•зҗҶпјҢжҲ–иҖ…жҳҜеңЁе®№еҷЁе…§йҒӢиЎҢгҖӮиҖҢжҜҸеҖӢJobsдёӯжңүеҢ…еҗ«дәҶдёҖеҖӢжҲ–еӨҡеҖӢ Steps пјҢжңғеҹ·иЎҢдҪ е®ҡзҫ©зҡ„и…іжң¬жҲ–жҳҜжҹҗеҖӢ ActionпјҢActionжҳҜGithub Actionз•¶дёӯзҡ„жңҖе°Ҹе–®дҪҚпјҢжҳҜдёҖеҖӢеҸҜйҮҚиӨҮдҪҝз”Ёзҡ„ж“ҙеұ•пјҢйҖҡеёёжҳҜжҹҗдәӣйҖҡз”ЁжҢҮд»Ө(?рҹҳҘ)

рҹ”ҖWorkflows

  • дёҖеҖӢеҸҜиЁӯе®ҡдҫҶиҮӘеӢ•еҢ–жөҒзЁӢзҡ„дёҖеҖӢжҲ–еӨҡеҖӢJobsиў«зЁұдҪңзӮәWorkflowпјҢеҸҜд»ҘйҖҸйҒҺж’°еҜ«YAMLжӘ”дҫҶе®ҡзҫ©Workflow,дёҰеңЁEventзҷјз”ҹжҷӮиў«и§ёзҷјпјҢд№ҹеҸҜд»ҘжүӢеӢ•и§ёзҷјгҖӮ
  • Workflowиў«е®ҡзҫ©еңЁRepoдёӯзҡ„ .github/workflows и·Ҝеҫ‘еә•дёӢ
  • дёҖеҖӢRepoеҸҜд»ҘжңүеӨҡеҖӢWorkflowsпјҢдёҰиІ иІ¬еҹ·иЎҢеҗ„зЁ®дёҚеҗҢзҡ„д»»еӢҷ
    • еғҸжҳҜпјҢдёҖеҖӢз”Ёж–јBuildи·ҹTestзҡ„Workflow
    • еҸҰдёҖеҖӢеүҮжҳҜеңЁReleaseзҷјеёғеҫҢйғЁзҪІжҮүз”ЁеҲ°з”ҹз”ўз’°еўғзҡ„Workflow

  • pull request
  • open issues
  • psuhes a commit to repo вҖҰetc

дёҖзө„Stepsиў«зЁұдҪңJob,дёҰдё”еҹ·иЎҢеңЁRunnerдёҠпјҢжҜҸеҖӢStepsеҸҜиғҪжңғжҳҜдёҖеҖӢShell ScriptжҲ–жҳҜActionиў«еҹ·иЎҢгҖӮStepsжңғдҫқеәҸеҹ·иЎҢпјҢдё”з”ұж–јжҜҸеҖӢStepsжҳҜеңЁзӣёеҗҢзҡ„RunnerдёҠиў«еҹ·иЎҢпјҢеӣ жӯӨеҸҜд»ҘеңЁдёҚеҗҢStepsд№Ӣй–“е…ұдә«зӣёеҗҢиіҮж–ҷгҖӮ иҲүдҫӢпјҡ дёҖеҖӢstepе»әзҪ®е®ҢжҮүз”ЁеҫҢеҸҰдёҖеҖӢstepдҫҶжё¬и©ҰжҮүз”ЁгҖӮ

дҪ д№ҹеҸҜд»ҘиЁӯе®ҡJobsзҡ„дҫқиіҙй …пјҢдёҖж—ҰдёҖеҖӢJobsдҫқиіҙеҸҰдёҖеҖӢJobпјҢеүҮжңғзӯүеҫ…еҸҰдёҖеҖӢJobе®ҢжҲҗеҫҢжүҚй–Ӣе§Ӣеҹ·иЎҢиҮӘе·ұзҡ„д»»еӢҷгҖӮ

рҹ“‘Actions

ActionжҳҜдёҖеҖӢе°ҲзӮәGithub Actionе№іеҸ°иЁӯиЁҲзҡ„зЁӢеәҸпјҢеҸҜеҹ·иЎҢиӨҮйӣңгҖҒй«ҳй »зҺҮдё”йҮҚиӨҮзҡ„е·ҘдҪңгҖӮ
йҖҸйҒҺActionеҸҜд»Ҙжёӣе°‘еҮәзҸҫеңЁworkflowжӘ”жЎҲдёӯеӨ§йҮҸйҮҚиӨҮзҡ„зЁӢејҸзўјгҖӮ
дёҖеҖӢActionеҸҜд»ҘеҒҡеҲ°еғҸжҳҜеҫһGithubз•¶дёӯжӢүеҸ–repositoryпјҢдёҰзӮәжүҖе»әзҪ®зҡ„з’°еўғиЁӯе®ҡжӯЈзўәзҡ„е·Ҙе…·йӣҶпјҢдёҰиЁӯе®ҡеҘҪиҲҮдҪ зҡ„йӣІз«ҜдҫӣжҮүз”Ёе•Ҷд№Ӣй–“зҡ„иә«еҲҶй©—иӯүгҖӮ

иЁӯе®ҡAction:https://docs.github.com/en/actions/creating-actions

Workflows File йҖҗиЎҢи§ЈйҮӢ

еҸғиҖғ: Workflow Syntax

  • name : жҢҮе®ҡWorkflowеҗҚзЁұпјҢдёҰжңғеҮәзҸҫеңЁRepositoryдёӯзҡ„Action Tab
  • run-name : жҜҸж¬ЎйҒӢиЎҢи©ІWorkflowжҷӮжңғеҮәзҸҫзҡ„ж–Үеӯ—иЁҠжҒҜпјҢйҖҷйӮҠжҢҮе®ҡ repo username + вҖңis learning Github ActionsвҖқ

  • on : жҢҮе®ҡжңғи§ёзҷј Workflows зҡ„иЎҢзӮәпјҢйҖҷиЈЎдҪҝз”Ё push eventпјҢжүҖд»ҘдҫқдҪҶжңүдәә pushжӣҙеӢ•еҲ° repositoryпјҢеүҮжңғи§ёзҷј Workflowsзҡ„еҹ·иЎҢ
  • jobs : е°Ү Workflowsз•¶дёӯзҡ„jobзө„еҗҲеңЁдёҖиө·
  • check-bats-version : иҮӘе®ҡзҫ©Jobзҡ„еҗҚзЁұ
  • runs-on: иЁӯе®ҡJobеҹ·иЎҢеңЁOSзӮәжңҖж–°зүҲжң¬зҡ„Ubuntu Linuxзҡ„RunnerдёҠпјҢжӯӨжҢҮе®ҡжңғи§ёзҷјHostedжүҖиЁ—з®Ўзҡ„иҷӣж“¬ж©ҹиў«е»әз«ӢгҖӮ
  • steps: е°Ү check-bats-version job еә•дёӢзҡ„жүҖжңүstepзө„еҗҲеҶҚдёҖиө·
  • uses:гҖҖз”ЁдҫҶжҢҮе®ҡstepзҡ„е‘Ҫд»ӨжҲ–иЎҢзӮә
    • actions/checkout@v3: д»ЈиЎЁиҰҒеҹ·иЎҢv3зүҲжң¬зҡ„actions/checkoutиЎҢзӮәпјҢжӯӨжӯҘй©ҹжңғе°ҮдҪ зҡ„repo checkoutеҲ°RunnerдёҠпјҢдёҰе…ҒиЁұдҪ йҮқе°ҚдҪ зҡ„зЁӢејҸйҖІиЎҢж“ҚдҪңпјҢжҜҸз•¶дҪ зҡ„Workflowе°ҮйҮқе°ҚRepositoryзҡ„зЁӢејҸйҒӢиЎҢжҷӮпјҢйғҪжҮүи©ІдҪҝз”Ё checkout ж“ҚдҪң
    • actions/setup-node@v3: жӯӨжӯҘй©ҹжңғе®үиЈқжҢҮе®ҡзүҲжң¬зҡ„NodeJS(жӯӨзҜ„дҫӢдёӯиҰ–зүҲжң¬1)
  • run : runжҢҮд»Өжңғе‘ҠиЁҙ jobеңЁRunnerдёӯеҹ·иЎҢзү№е®ҡжҢҮд»ӨпјҢжӯӨзҜ„дҫӢзөӮз©¶жҳҜе‘ҠиЁҙJobеңЁRunnerдёӯеҹ·иЎҢ npm install batsпјҢд»ҘеҸҠдёӢдёҖжӯҘеүҮжҳҜеҺ»жҹҘзңӢ batsзүҲжң¬
