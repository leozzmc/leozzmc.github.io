  4. еӣһеӮіhead

иӘӘжҳҺ

йҰ–е…ҲжҳҜеҲқе§ӢеҢ–пјҢansHead з”ЁдҫҶдҝқеӯҳзөҗжһңlistзҡ„headпјҢcurrent з”ЁдҫҶйҒҚжӯ·listпјҢprevious з”ЁдҫҶиЁҳйҢ„з•¶еүҚзҜҖй»һзҡ„еүҚдёҖеҖӢзҜҖй»һгҖӮжҺҘи‘—з•¶ current!= NULL зҡ„жҷӮеҖҷпјҢжңғеҺ»еҫӘз’°йҒҚжӯ·зҜҖй»һпјҢйҖҷиЈЎе°Үй ӯзҜҖй»һд»ҘеҸҠе…¶д»–зҜҖй»һеҲҶй–Ӣиҷ•зҗҶпјҢеӣ зӮәжҲ‘еёҢжңӣеҰӮжһңжңүйҮҚиӨҮпјҢеҲӘйҷӨзҡ„йғҪжҳҜз¬¬2еҖӢд»ҘеҫҢзҡ„зҜҖй»һпјҢеӣ жӯӨиө°еҲ°й ӯзҜҖй»һеғ…йңҖиҰҒжӣҙж–°жҢҮжЁҷе°ұеҘҪпјҢиө°еҲ°е…¶д»–зҜҖй»һжҷӮпјҢе°ұйңҖиҰҒжӘўжҹҘеүҚдёҖеҖӢзҜҖй»һи·ҹзҸҫеңЁзҜҖй»һжҳҜеҗҰдёҖжЁЈ (previous->val == current->val && current->next != NULL) еҰӮжһңдёҖжЁЈпјҢиҰҒеҒҡзҡ„дәӢжғ…жңүдёүд»¶:

  1. е»әз«ӢиҮЁжҷӮжҢҮжЁҷпјҢз”ЁдҫҶжҢҮеҗ‘ current зҡ„дёӢдёҖеҖӢзҜҖй»һ
  2. е°Ү current жҢҮеҗ‘е…¶дёӢдёҖеҖӢзҜҖй»һ
  3. жӣҙж–° previous->next зӮә current

еҰӮжһңжңүеӨҡеҖӢйҮҚиӨҮзҜҖй»һпјҢе°ұеҸҜд»ҘеңЁ while иҝҙеңҲе…§дёҖдҪөеҲӘйҷӨ

жҷӮй–“иӨҮйӣңеәҰ

  • еӨ–еұӨ while иҝҙеңҲ: йҖҷеҖӢиҝҙеңҲйҒҚжӯ·ж•ҙеҖӢlistпјҢжҜҸеҖӢзҜҖй»һиЁӘе•ҸдёҖж¬ЎпјҢеӣ жӯӨзёҪе…ұеҹ·иЎҢ n ж¬ЎпјҢе…¶дёӯ n жҳҜlistзҡ„зҜҖй»һж•ёгҖӮ

  • е…§еұӨ while иҝҙеңҲ: йҖҷеҖӢиҝҙеңҲеңЁйҒҮеҲ°йҮҚиӨҮзҡ„зҜҖй»һжҷӮжңғеҹ·иЎҢпјҢдёҰдё”жңҖеӨҡд№ҹеҸӘжңғеҹ·иЎҢ n ж¬ЎпјҲеӣ зӮәжҜҸж¬Ўе…§еұӨиҝҙеңҲеҹ·иЎҢйғҪжңғе°Ү current жҺЁйҖІиҮіе°‘дёҖеҖӢзҜҖй»һпјҢеҫһиҖҢдҝқиӯүе…§еұӨиҝҙеңҲзҡ„зёҪеҹ·иЎҢж¬Ўж•ёдёҚжңғи¶…йҒҺзҜҖй»һж•ё nпјүгҖӮ

  • е…¶д»–ж“ҚдҪң: е…§йғЁзҡ„жўқд»¶еҲӨж–·е’ҢжҢҮйҮқж“ҚдҪңйғҪжҳҜеёёж•ёжҷӮй–“зҡ„ж“ҚдҪң

  • йЎҚеӨ–и®Ҡж•ё: еҸӘдҪҝз”ЁдәҶе°‘йҮҸзҡ„йЎҚеӨ–и®Ҡж•ё (ansHeadпјҢcurrentпјҢprevious е’Ң temp)пјҢйҖҷдәӣи®Ҡж•ёзҡ„ж•ёйҮҸиҲҮlistзҡ„еӨ§е°Ҹз„Ўй—ңпјҢжҳҜеёёж•ёеҖӢж•ёгҖӮ
  • дёҰжІ’жңүдҪҝз”ЁйЎҚеӨ–зҡ„еӢ•ж…ӢиЁҳжҶ¶й«”пјҢеүөе»әдәҶиҮЁжҷӮtemp дёҰдёҚеҪұйҹҝж•ҙй«”зҡ„з©әй–“иӨҮйӣңеәҰгҖӮ
