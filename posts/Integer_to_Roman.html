|й–ұи®ҖйҮҸ:

йЎҢзӣ®ж•ҳиҝ°


  • йЎҢзӣ®йӣЈеәҰпјҡMedium
  • йЎҢзӣ®ж•ҳиҝ°пјҡйЎҢзӣ®дё»иҰҒйңҖжұӮжҳҜе°Үијёе…Ҙзҡ„ж•ҙж•ё num иҪүжҸӣжҲҗе°ҚжҮүзҡ„зҫ…йҰ¬ж•ёеӯ—з¬ҰиҷҹпјҢдёҰзөҰе®ҡдёҖеҖӢе°ҚжҮүиЎЁпјҢеңЁиҪүжҸӣйҒҺзЁӢжңүе№ҫй …иҪүжҸӣиҰҸеүҮпјҡ
    • зҫ…йҰ¬ж•ёеӯ—жҳҜйҖҡйҒҺеҫһжңҖй«ҳдҪҚеҲ°жңҖдҪҺдҪҚе°Үе°Ҹж•ёдҪҚеҖјзҡ„иҪүжҸӣйҖЈжҺҘиө·дҫҶеҪўжҲҗзҡ„
    • еҰӮжһңи©ІеҖјдёҚд»Ҙ4жҲ–9й–Ӣй ӯ,еүҮйҒёж“ҮеҸҜд»Ҙеҫһијёе…ҘдёӯжёӣеҺ»зҡ„жңҖеӨ§еҖјзҡ„з¬Ұиҷҹ,е°Үи©Із¬Ұиҷҹйҷ„еҠ еҲ°зөҗжһңдёӯ,жёӣеҺ»е…¶еҖј,з„¶еҫҢе°Үеү©йӨҳйғЁеҲҶиҪүжҸӣзӮәзҫ…йҰ¬ж•ёеӯ—гҖӮ
    • еҰӮжһңи©ІеҖјд»Ҙ4жҲ–9й–Ӣй ӯ,еүҮдҪҝз”ЁиЎЁзӨәеҫһдёӢдёҖеҖӢз¬ҰиҷҹдёӯжёӣеҺ»дёҖеҖӢз¬Ұиҷҹзҡ„жёӣжі•еҪўејҸ,дҫӢеҰӮ,4жҳҜ 5(V)жёӣеҺ»1(I):IV,иҖҢ9жҳҜ10(X)жёӣеҺ»1(I):IXгҖӮеҸӘдҪҝз”Ёд»ҘдёӢжёӣжі•еҪўејҸ:4(IV)гҖҒ9(IX)гҖҒ40(XL)гҖҒ90(XC)гҖҒ400(CD)е’Ң900(CM)гҖӮ
    • еҸӘжңү10зҡ„ж¬Ўж–№(IгҖҒXгҖҒCгҖҒM)еҸҜд»ҘжңҖеӨҡйҖЈзәҢйҷ„еҠ 3ж¬Ўд»ҘиЎЁзӨә10зҡ„еҖҚж•ёгҖӮдёҚиғҪеӨҡж¬Ўйҷ„еҠ 5(V)гҖҒ50(L)жҲ–500(D)гҖӮеҰӮжһңйңҖиҰҒйҷ„еҠ 4ж¬Ўз¬Ұиҷҹ,йңҖдҪҝз”Ёжёӣжі•еҪўејҸ
  • йҷҗеҲ¶пјҡ 1 <= num <= 3999

и§Јжі•

дёҖй–Ӣе§Ӣзҡ„жғіжі•

жҲ‘е…¶еҜҰдёҖй–Ӣе§ӢеҒҸеҗ‘жҡҙеҠӣи§ЈпјҢе°ұж №ж“ҡжўқд»¶еҺ»еҖӢеҲҘеҲӨж–·пјҢз„¶еҫҢеҫһж•ёеӯ—зҡ„й«ҳдҪҚж•ёй–Ӣе§ӢеҲӨж–·дёҰиҪүжҸӣз¬ҰиҷҹпјҢе·Із¶“иҪүйҒҺзҡ„ж•ёеӯ—е°ұжёӣжҺү

жҲ‘зҡ„и§Јжі•

иӘӘжҳҺ

  • еҫһжңҖй«ҳдҪҚзҡ„ 1000 - 3000 й–Ӣе§Ӣиҷ•зҗҶпјҢеҲӨж–·жңҖй«ҳдҪҚзҡ„ж•ёеӯ—пјҢдёҰдё”е°Ү results ж·»еҠ зӣёе°ҚжҮүеҖӢж•ёзҡ„ M
  • жҺҘи‘—еҲӨж–·зҷҫдҪҚж•ёпјҢйҰ–е…Ҳиҷ•зҗҶ400 д»ҘеҸҠ 900 еӨ–зҡ„зӢҖжіҒпјҢдёҰдё”еҲҶеҲҘиҖғж…® >500 д»ҘеҸҠ < 500 е’Ң = 500 зҡ„дёүзЁ®зӢҖжіҒпјҢеҰӮжһңжңү500еүҮйңҖиҰҒйЎҚеӨ–еңЁ results ж·»еҠ  D дёҰдё”num йңҖиҰҒйЎҚеӨ–жүЈжҺү500
  • еҲҶеҲҘиҷ•зҗҶ 400 е’Ң 900зҡ„зӢҖжіҒ
  • жҺҘи‘—иҷ•зҗҶеҚҒдҪҚж•ёпјҢдёҖжЁЈе…Ҳиҷ•зҗҶ 40, 90еӨ–зҡ„зӢҖжіҒпјҢдёҰдё”еҲҶеҲҘиҖғж…® >50, <50 е’Ң =50 дёүзЁ®зӢҖжіҒпјҢиӢҘжңү50еүҮйңҖйЎҚеӨ–еңЁ results ж·»еҠ  L,дёҰдё”resultsиҰҒжүЈжҺү50
  • жңҖеҫҢиҷ•зҗҶеҖӢдҪҚж•ё
  • еӣһеӮізөҗжһң

еҹ·иЎҢзөҗжһң

е…¶д»–еҒҡжі•

еҸҰдёҖзЁ®еҒҡжі•е°ұжҳҜйҖҸйҒҺеҜҰдҪң Hash TableпјҢйҖҷиЈЎеҸҜд»ҘйҖҸйҒҺ C++ STL дёӯзҡ„ pair е®№еҷЁеҺ»е»әз«Ӣе°ҚжҮүзҡ„еӯ—е…ёпјҢдёҰдё”йҖҸйҒҺиҝҙеңҲеңЁеҲӨж–·ж•ёеӯ—жҳҜеҗҰеӨ§ж–јеӯ—е…ёдёӯзҡ„keyж¬„дҪҚ(ж•ёеӯ—)пјҢеҰӮжһңеӨ§ж–јзӯүж–ји©ІKeyпјҢеүҮе°ұеңЁresults ж·»еҠ е°ҚжҮүзҡ„з¬ҰиҷҹпјҢдёҰдё”е°Үnum жүЈйҷӨе·Із¶“ж·»еҠ зҡ„ж•ёеӯ—
еҰӮжӯӨиҝӯд»ЈдёӢдҫҶпјҢеҫ—еҮәзҡ„зөҗжһңд№ҹжңғжҳҜжҲ‘еҖ‘иҰҒзҡ„зҫ…йҰ¬еӯ—дёІгҖӮ

еңЁ Pair дёӯпјҢеҸҜд»ҘйҖҸйҒҺ first д»ЈиЎЁз¬¬дёҖеҖӢе…ғзҙ пјҢд№ҹе°ұжҳҜkeyпјҢsecond еҸҜд»ЈиЎЁз¬¬дәҢеҖӢе…ғзҙ пјҢд№ҹе°ұжҳҜ value

еҹ·иЎҢзөҗжһң

иӨҮйӣңеәҰеҲҶжһҗ

йҖҷиЈЎеҸҜд»ҘеҲҶжһҗжңҖдҪіи§Јзҡ„иӨҮйӣңеәҰ

жҷӮй–“иӨҮйӣңеәҰ

йҖҷж®өд»Јзўјзҡ„жҷӮй–“иӨҮйӣңеәҰзӮә $O(1)$

еӨ–еұӨзҡ„ for еҫӘз’°йҒҚжӯ·дәҶ RomanDict еҗ‘йҮҸ,еҗ‘йҮҸзҡ„еӨ§е°ҸжҳҜеӣәе®ҡзҡ„ 13,жүҖд»ҘеҫӘз’°еҹ·иЎҢзҡ„ж¬Ўж•ёжҳҜеёёж•ёжҷӮй–“ $O(1)$
е…§еұӨзҡ„ while еҫӘз’°зҡ„еҹ·иЎҢж¬Ўж•ёеҸ–жұәж–јијёе…Ҙж•ёеӯ— num зҡ„еӨ§е°ҸгҖӮз„¶иҖҢ,з”ұж–јијёе…ҘзҜ„еңҚжҳҜжңүйҷҗзҡ„ (1 <= num <= 3999),еӣ жӯӨе…§еұӨеҫӘз’°зҡ„еҹ·иЎҢж¬Ўж•ёд№ҹжҳҜжңүйҷҗзҡ„гҖӮ
еңЁжңҖеЈһзҡ„жғ…жіҒдёӢ,е…§еұӨеҫӘз’°йңҖиҰҒеҹ·иЎҢзҡ„ж¬Ўж•ёиҲҮијёе…Ҙж•ёеӯ—зҡ„дҪҚж•ёжҲҗжӯЈжҜ”гҖӮз”ұж–јијёе…ҘзҜ„еңҚжңүйҷҗ,дҪҚж•ёд№ҹжҳҜжңүйҷҗзҡ„,еӣ жӯӨе…§еұӨеҫӘз’°зҡ„жҷӮй–“иӨҮйӣңеәҰжҳҜеёёж•ёжҷӮй–“ $O(1)$
зёҪзҡ„дҫҶиӘӘ,з„Ўи«–ијёе…Ҙж•ёеӯ—жҳҜеӨҡе°‘,зЁӢејҸзўјзҡ„еҹ·иЎҢжҷӮй–“йғҪжҳҜеӣәе®ҡзҡ„,еӣ жӯӨжҷӮй–“иӨҮйӣңеәҰзӮәеёёж•ёжҷӮй–“ $O(1)$

з©әй–“иӨҮйӣңеәҰ

йҖҷж®өд»Јзўјзҡ„з©әй–“иӨҮйӣңеәҰзӮә $O(1)$

зЁӢејҸзўјдёӯдҪҝз”ЁдәҶдёҖеҖӢеӣәе®ҡеӨ§е°Ҹзҡ„еҗ‘йҮҸ RomanDict дҫҶеӯҳе„Ізҫ…йҰ¬ж•ёеӯ—еҸҠе…¶е°ҚжҮүзҡ„еҖј,е®ғзҡ„еӨ§е°ҸзӮә 13,дёҚйҡЁијёе…Ҙзҡ„и®ҠеҢ–иҖҢи®ҠеҢ–гҖӮ
ијёеҮәеӯ—з¬ҰдёІ results зҡ„еӨ§е°ҸжңҖеӨҡзӮә 15 еҖӢеӯ—з¬Ұ(е°ҚжҮүжңҖеӨ§зҡ„зҫ…йҰ¬ж•ёеӯ—вҖқMMMCMXCIXвҖқ),йҖҷд№ҹжҳҜдёҖеҖӢеӣәе®ҡзҡ„еёёж•ёз©әй–“гҖӮ
еӣ жӯӨ,з„Ўи«–ијёе…Ҙж•ёеӯ—жҳҜеӨҡе°‘,д»ЈзўјжүҖйңҖзҡ„йЎҚеӨ–з©әй–“йғҪжҳҜеӣәе®ҡзҡ„,еӣ жӯӨз©әй–“иӨҮйӣңеәҰзӮәеёёж•ёз©әй–“ $O(1)$

иҮіж–јеҺҹжң¬жҡҙеҠӣи§Јзҡ„иӨҮйӣңеәҰеңЁпјҢијёе…ҘзҜ„еңҚжҸҗй«ҳзҡ„жҷӮеҖҷпјҢиӨҮйӣңеәҰд№ҹжңғйҡЁд№ӢеўһеҠ пјҢеҰӮжһңйңҖиҰҒж”ҜжҢҒжӣҙеӨ§зҜ„еңҚзҡ„ијёе…Ҙ,з•¶ијёе…Ҙж•ёеӯ— num зҡ„еӨ§е°ҸеўһеҠ жҷӮ,йҖҷж®өд»Јзўјзҡ„жҷӮй–“иӨҮйӣңеәҰе°Үжңғи®ҠзӮә $O(logn)$,е…¶дёӯ n жҳҜијёе…Ҙж•ёеӯ—зҡ„еӨ§е°ҸгҖӮ
